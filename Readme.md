ROBOTRADING: Construyendo robots de trading con Software Libre
========================================================

En esta conferencia se muestra la importancia de los robots de trading en la economía actual, así como las oportunidades para los programadores e ingenieros en empleabilidad a nivel mundial, también se explica un modelo para su creación y la aplicación de algoritmos en lenguajes de programación gratuitos. Además, se mostrará a SANTI (Sistema Automatizado Neural Inteligente) un desarrollo local con aplicaciones de inteligencia artificial funcionando en tiempo real.
